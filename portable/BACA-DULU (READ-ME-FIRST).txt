 COEBOT - Portable Edition
 No Python, no Git, no terminal. Everything is in this folder.
============================================================

HOW TO START (2 things, one time only)
--------------------------------------

1. DOWNLOAD THE AI MODEL (one time, ~5.6 GB)

   Open this link in your browser - the download starts instantly:

   https://huggingface.co/unsloth/Qwen3.5-9B-GGUF/resolve/main/Qwen3.5-9B-UD-Q4_K_XL.gguf?download=true

   When it finishes, MOVE the downloaded file
   (Qwen3.5-9B-UD-Q4_K_XL.gguf) from Downloads into the
   "models" folder inside this folder.

   Have 32 GB RAM? You can use the bigger, smarter model instead:
   https://huggingface.co/unsloth/Qwen3-30B-A3B-Instruct-2507-GGUF/resolve/main/Qwen3-30B-A3B-Instruct-2507-Q4_K_M.gguf?download=true

2. DOUBLE-CLICK  start_coebot.bat

   A black window opens, then your browser opens at
   http://127.0.0.1/ - register a username and password
   (stored only on this computer) and start chatting.

   To stop COEBOT: close the black window.
   Next time: just double-click start_coebot.bat again.

------------------------------------------------------------
BAHASA INDONESIA
------------------------------------------------------------

1. UNDUH MODEL AI (sekali saja, ~5,6 GB)

   Buka tautan di atas (nomor 1) - unduhan langsung mulai.
   Setelah selesai, PINDAHKAN file .gguf dari folder Downloads
   ke folder "models" di dalam folder ini.

2. KLIK DUA KALI  start_coebot.bat

   Jendela hitam terbuka, lalu browser terbuka otomatis di
   http://127.0.0.1/ - daftar username dan password (tersimpan
   hanya di komputer ini), lalu mulai chat.

   Untuk berhenti: tutup jendela hitam.
   Lain kali: cukup klik dua kali start_coebot.bat lagi.

------------------------------------------------------------
REQUIREMENTS / SYARAT
------------------------------------------------------------
- Windows 10 or 11, 64-bit
- 16 GB RAM (minimum) - use the 9B model
  32 GB RAM - you may use the 30B model
- ~10 GB free disk space (plus the model file)
- Everything runs 100% offline. No data leaves this computer.
